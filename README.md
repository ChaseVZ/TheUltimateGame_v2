# TheUltimateGame_v2

/// IntelliJ configuration needed ///

type - Application
program arguments - '-fastest' || -'faster' || -'fast'
set main class - Virtual World

/// Project Settings Needed ///

Set 'src' folder to be 'sources'
add 'processing-experimental.jar' to your set of libraries (not Global Libraries)

/// Other ///

if you are getting errors, make sure 'hamcrest-core-1.3' and 'junit-4.12' are installed on your machine and 
logged as Global libraries in your project structure.

you can also make 'processing-experimental.jar' a Global Library. Unsure if this is necessary
